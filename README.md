# UNT Unreal Engine TTS Pipeline

**Tools:** Unreal Engine 5.3 · Blueprint Visual Scripting · RESTful API Integration · JSON Parsing · Python · C++

## Overview
This repository contains reference scripts and snippets from a modular **Text-to-Speech (TTS) pipeline** built in **Unreal Engine 5.3** for the **University of North Texas (UNT)** research program.  
The system converts AI-generated or API-based text into lifelike speech using the **native Media Framework**, eliminating the need for third-party plugins.
